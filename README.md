# Office Suggestion Box

This is the uploadable project folder for your website.

## Files that should be in GitHub

- package.json
- tsconfig.json
- tsconfig.app.json
- tsconfig.node.json
- vite.config.ts
- index.html
- public/logo.png
- src/main.tsx
- src/App.tsx
- src/styles.css

## Environment variables for Vercel

Add these in Vercel later:

- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY

## Local note

If these environment variables are missing, the site still works in demo mode.
